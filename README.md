# DS Helper

**DS Helper** ist eine Sammlung von Schnellleistenskripten für **Die Stämme**, die verschiedene Spielansichten um nützliche Komfortfunktionen erweitert.

Ziel des Projekts ist es, wiederkehrende Aufgaben zu vereinfachen, Informationen übersichtlicher darzustellen und den Spielalltag komfortabler zu gestalten.

Alle Skripte werden kontinuierlich weiterentwickelt und über GitHub versioniert.

---

# Verfügbare Skripte

| Kategorie | Skript | Version | Status |
|-----------|---------|:-------:|:------:|
| Produktion | Freie BHP | **v1.1.0** | ✅ |

---

# Installation

Die Skripte werden als Schnellleistenskripte in **Die Stämme** verwendet.

Beispiel:

```javascript
javascript:$.getScript("https://cdn.jsdelivr.net/gh/Rincewind610/ds-helper-scripts@v1.1.0/freie-bhp.js");
```

---

# Versionierung

Jede veröffentlichte Version erhält einen eigenen GitHub-Release.

Die aktuelle Entwicklung erfolgt über den `main`-Branch, stabile Versionen werden als Release veröffentlicht.
